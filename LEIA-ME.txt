Intruções para compilação

REQUISITOS:

O.S: Ubuntu
gcc version 7.3.0
makefile
gtest version 1.8.1
cpplint
gcov version 7.3.0
cppcheck
valgrind

INSTALAÇÕES NECESSÁRIAS:

GCC,MAKEFILE,CPPCHECK,GCOV:

sudo apt-get update
sudo apt-get install build-essential

GTEST:

sudo apt-get install cmake # install cmake
cd /usr/src/gtest
sudo cmake CMakeLists.txt
sudo make
sudo cp *.a /usr/lib

CPPLINT:

sudo apt-get install python-pip  
sudo pip install cpplint  

CPPCHECK:

sudo apt-get update
sudo apt-get install cppcheck

VALGRIND:

sudo apt-get update
sudo apt-get install valgrind

COMPILAÇÃO:

Abra o terminal e execute os seguintes comandos:

cd Trabalho-MP-03/code/source

make

Para ver os resultados dos testes, verificar cppcheck e valgrind:

valgrind ./testa_pilha && cppcheck --enable=warning testa_pilha.c pilha.c

#Para ver os resultado do GCOV execute os seguintes comandos:

cd obj

gcov testa_pilha.gcno

Para mudar de implementaçao para STACK-LINKED basta comentar a parte STACK-ARRAY e vice-versa.
